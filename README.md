# Конструктор Лендингов

## Описание
Полнофункциональный конструктор лендингов с drag-and-drop интерфейсом, настройками дизайна и экспортом готового HTML/CSS кода.

## Основные возможности

### 1. Библиотека блоков
- **Шапка и навигация**: Header, Navbar
- **Hero блоки**: Hero Banner, Hero Video, Hero Split
- **Контент**: Features, Features Grid, About, Statistics
- **Отзывы**: Testimonials, Team
- **Цены и CTA**: Pricing, Call to Action
- **Формы**: Contact Form, Newsletter
- **Футер**: Footer

### 2. Drag-and-Drop функциональность
- Перетаскивание блоков из панели на canvas
- Перемещение блоков вверх/вниз
- Удаление блоков
- Визуальная подсветка при перетаскивании

### 3. Панель настроек
- Изменение фонового цвета блока
- Изменение цвета текста
- Настройка отступов
- Редактирование всех текстов в блоке

### 4. Экспорт
- Генерация чистого HTML/CSS кода
- Без комментариев
- Без смайликов
- Готовый к использованию файл

## Технологии
- HTML5
- CSS3 (с градиентами и современными стилями)
- Vanilla JavaScript
- Font Awesome для иконок

## Структура файлов
- `index.html` - главная страница конструктора
- `style.css` - стили интерфейса конструктора
- `script.js` - функциональность (drag-and-drop, настройки, экспорт)


## Использование
1. Выберите нужные блоки из левой панели
2. Перетащите их на центральную область (canvas)
3. Кликните на блок для редактирования
4. Настройте цвета, отступы и тексты в правой панели
5. Нажмите "Экспорт HTML/CSS" для скачивания готового лендинга
